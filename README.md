# Kervin Kyle Lubayan — Travel Journal

A personal travel, food, and photography portfolio built as a digital magazine-style journal. Features stories from the NAIA runway, Filipino food culture, and photo essays — submitted as a course requirement for BSTM 603, Tourism Management.

---

## Tech Stack

- **HTML** — semantic page structure and modal/overlay system
- **CSS** — custom design system with CSS variables, responsive grid layouts, and animations
- **JavaScript** — scroll spy, before/after image slider, mobile nav, highlight toolbar, and entrance animations
